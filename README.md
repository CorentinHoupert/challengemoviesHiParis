This repository is related to a challenge for a research engineer with Hi!Paris.
https://hi-paris.fr/

The aim of the challenge is to predict the genre of a film based only on its poster.
The dataset is in the file movies.csv
It is required to download the posters on your machine, this can be done with telechargementdesaffiches.ipynb

An inspection of the dataset can be done using dataanalysis.ipynb

The classification of the posters through their genre is made using a CNN predict a multi-label for each movie. This can be done with classification.ipynb

ChatGPT GPT-5.3-mini was largely used in this project.
